Minecraft Commands

This page lists commands that don't require cheats or operator permissions in Minecraft: Bedrock Edition version 1.21.131.

Command Arguments What It Does
/? [command] Alias of /help
/help [command] Shows command help
/list None Lists online players
/me <greedy phrase string> Used for sending a message written in third-person
/msg <player> <greedy phrase string> Sends a private message to a specific player
/tell <player> <greedy phrase string> Alias of /msg
/w <player> <greedy phrase string> Alias of /msg
